### 2.1 AIPanel 组件结构

```
AIPanel (ai_panel.ts)
│
├── State
│   ├── input: string                    # 当前输入
│   ├── messages: Message[]              # 聊天消息历史
│   ├── isLoading: boolean               # 加载状态
│   ├── backendTraceId: string | null    # 后端 trace ID
│   ├── settings: AISettings             # AI 配置
│   └── sessions: AISession[]            # 会话列表
│
├── UI Components
│   ├── Header                           # 标题 + 设置按钮
│   ├── SessionSidebar                   # 会话列表侧边栏
│   ├── MessagesContainer                # 消息显示区域
│   │   ├── UserMessage                  # 用户消息气泡
│   │   └── AssistantMessage             # AI 回复气泡
│   │       ├── Markdown Content         # 格式化文本
│   │       ├── SqlResultTable           # SQL 结果表格
│   │       └── ReportLink               # HTML 报告链接
│   └── InputArea                        # 输入框 + 发送按钮
│
└── Event Handlers
    ├── sendMessage()                    # 发送消息
    ├── handleCommand()                  # 处理 /命令
    ├── handleChatMessage()              # 处理自然语言
    ├── listenToSSE()                    # SSE 流监听
    └── handleSSEEvent()                 # SSE 事件处理
```

```bash
# 使用旧架构 (默认)

USE_MASTER_ORCHESTRATOR=true npm run dev
```
