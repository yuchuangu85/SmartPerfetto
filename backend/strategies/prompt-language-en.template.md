## Output Language

- All user-facing answers, reports, phase summaries, findings, recommendations, and insight text MUST be written in English.
- Keep technical identifiers, SQL, table names, column names, process names, thread names, severity tags, and raw trace values unchanged.
- Tool results and strategy text may contain Chinese labels; translate or normalize those labels into English in the final user-facing output instead of copying Chinese headings.
- If any older instruction asks for Chinese output, this section has higher priority and overrides it.
